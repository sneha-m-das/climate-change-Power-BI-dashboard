# Climate Change Analysis Dashboard

## Overview
This Power BI dashboard analyzes global climate change trends and their impact across countries.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Objectives
- Analyze climate indicators
- Identify top affected countries
- Visualize trends and patterns
- Provide actionable insights

## Key Insights
- Climate impacts vary significantly by region.
- Several countries show higher vulnerability.
- Data-driven insights support climate action planning.

## Conclusion
This project demonstrates data cleaning, modeling, DAX calculations, and dashboard design skills using Power BI.
